Pumpkin Face Projector 
================

## Dependencies
* ofxCv
* ofxOpenCv

## Commands
* 'n': next image
* 'p': previous image
* 'f': go to fullscreen
* 'c': calibrate

during calibration you can use the arrows to set the width and height of the projected images
